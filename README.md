chainer-svm
===========

This is an implementation of Support Vector Machines (SVM) using Chainer.

# Requirement

- [Chainer](https://github.com/chainer/chainer) (modified to chainer 7.7)

# Train & Test

```
$ python train_test.py --gpu 0
```
